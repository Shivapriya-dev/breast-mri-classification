# 🩺 Breast MRI Classification System using Deep Learning

A web-based AI application that classifies breast MRI images as **Benign** or **Malignant** using multiple state-of-the-art deep learning models.

The application is built using **TensorFlow**, **Streamlit**, and **Transfer Learning**, allowing users to compare predictions from different CNN architectures through an easy-to-use interface.

## 📌 Project Overview

Breast cancer is one of the leading causes of cancer-related deaths among women worldwide. Early diagnosis through MRI imaging significantly improves treatment outcomes.

This project applies Transfer Learning on pre-trained convolutional neural networks to automatically classify breast MRI images into:

- Benign
- Malignant

The application provides:

- Multiple AI model selection
- Confidence score prediction
- Simple web interface

> **Disclaimer:** This application is intended for academic and research purposes only and is **not a substitute for professional medical diagnosis.**

# 🧠 Deep Learning Models Used

The application supports four transfer learning models:

- EfficientNetB0
- ResNet50
- DenseNet121
- ConvNeXt Tiny

Each model can be selected independently from the interface for comparison.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/breast-mri-classification.git
```

Go into the project

```bash
cd breast-mri-classification
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application


# 📈 Transfer Learning

Instead of training models from scratch, this project uses pre-trained ImageNet weights and fine-tunes them for breast MRI classification.

Advantages include:

- Faster convergence
- Better accuracy
- Reduced training time
- Improved generalization
